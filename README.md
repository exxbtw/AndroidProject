# Визуальное программирование — Передача локации на imgui

Приложение реализует систему отслеживания параметров смартфона в реальном времени. Основные возможности:

* Получение точных координат (широта, долгота, высота, точность)
* Передача данных на удаленный C++ сервер через ZeroMQ
* Визуализация данных на бэкенде с использованием ImGui и ведение лога в JSON

---

## Файлы проекта (Android)

### Location2.kt

Класс активности для взаимодействия с пользователем:

* **Функционал:**
* Запрос разрешений на доступ к точной геолокации (`ACCESS_FINE_LOCATION`)
* Отображение текущих координат на экране
* Дублирующая отправка данных на сервер при изменении местоположения


### activity_location2.xml

Интерфейс мобильного приложения:

* `TextView` для вывода широты, долготы, высоты и времени последнего обновления
* Кнопка возврата в главное меню

---

## Файлы проекта (C++ Backend)

### main.cpp

Основной файл сервера мониторинга:

* **Многопоточность:**
* Основной поток: Рендеринг интерфейса (SDL2 + OpenGL + ImGui)
* Поток сервера: Прием сообщений через ZeroMQ (REP сокет)


* **Безопасность данных:**
* Использование `std::mutex` для предотвращения гонки потоков
* Флаг `volatile bool is_running` для запрета оптимизации, чтоб работало корректно в общем


* **Интерфейс (ImGui):**
* Окно с выводом текущих координат
* Система логирования: сохранение всех входящих пакетов в файл `location_log.json`
* Настройки масштабирования интерфейса и очистка логов
