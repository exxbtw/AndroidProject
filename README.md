# Визуальное программирование — Медиаплеер

Приложение реализует простой музыкальный плеер для Android с возможностью:
- Воспроизведения файлов из памяти устройства
- Переключения треков (Next / Previous)
- Паузы и продолжения воспроизведения
- Перемотки через `SeekBar`
- Отображения текущего времени и длительности трека
- Перемешивания списка треков (`Shuffle`)
- Сортировки треков по названию (`Sort`)
- Выбора трека из списка (`ListView`)

---

## Файлы проекта

### MediaPlayer.kt
Класс активности музыкального плеера:

- **Основные элементы управления:**
  - `MediaPlayer` — объект для воспроизведения звука
  - `ListView` — список доступных треков
  - `SeekBar` — позволяет перематывать трек
  - `Play/Pause`, `Next`, `Prev`, `Shuffle`, `Sort` — кнопки управления плеером

- **Структура:**
  - `onCreate()`:
    - Запрос разрешения на чтение аудиофайлов
    - Загрузка треков из папки `Music`
    - Настройка `ListView` и кнопок управления
  - `loadMusicFromStorage()` для получения списка `.mp3` файлов из памяти
  - `playTrack()` — воспроизведение выбранного трека
  - Обновление `SeekBar` и времени каждые 500 мс
  - `onResume()`:
      - Продолжает проигрывание при повторном открытии приложения из фона
  - `onPause()`:
      - Ограничивает работу приложения в фоне

### activity_media_player.xml
Разметка интерфейса:

- Используется — `ConstraintLayout`
- `TextView`:
  - Отображение имени текущего трека
  - Текущее время воспроизведения
  - Общая длительность трека
- `SeekBar` для перемотки
- Кнопки управления:
  - Play / Pause
  - Next
  - Previous
  - Shuffle
  - Sort
- `ListView` для отображения списка доступных треков
