# Визуальное программирование — Передача локации в фоне и CellInfo

Приложение реализует систему сбора и передачи телеметрии смартфона на удаленный сервер. Основные возможности:

* Фоновый мониторинг местоположения (GPS/GNSS).
* Сбор детальной информации о сотовых вышках.
* Передача данных по ZeroMQ в формате JSON.
* Отображение текущих координат в реальном времени в пользовательском интерфейсе.

---

## Файлы проекта (Android)

### Location2.kt

Класс активности (UI-слой) для взаимодействия с пользователем:

* **Запуск сервиса:** инициализация `TelephonyService` как Foreground-сервиса для работы в фоновом режиме.
* Визуализация: форматированный вывод широты, долготы и высоты в `TextView` с точностью до 6 знаков после запятой.



### TelephonyService.kt

Фоновый сервис:

* **Сетевое взаимодействие:**
* Использование библиотеки **ZeroMQ (ZMQ)**.
* Тип сокета: `REQ` (Request-Reply) для гарантированной доставки данных на сервер.


* **Сбор данных о связи:**
* Использование `TelephonyManager` для получения параметров активной соты.
* Поддержка различных стандартов: **LTE** (PCI, EARFCN, RSRP, RSRQ), **GSM** (LAC, CID) и **5G/NR** (NCI).

---

### activity_location2.xml

Интерфейс мобильного приложения:

* **Визуальные элементы:**
* Сетка из `TextView` для отображения:
* `tv_lat` / `tv_lon` — географические координаты.
* `tv_alt` — высота над уровнем моря.
* `tv_time` — временная метка (Timestamp) последнего пакета.

* `Button` (`back_to_main`) — навигация для возврата в главное меню.
